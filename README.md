# Stock Market Trend Analyzer

A Streamlit-based web application that analyzes stock trends, visualizes historical data, and predicts the next dayâ€™s closing price using a simple Linear Regression model.

# Features

- Fetch real-time stock data using yfinance
- Interactive dashboard built with Streamlit
- Stock price visualization using Matplotlib
- Trend detection using 20-day and 50-day moving averages
- Machine learning model (Linear Regression) to predict next-day closing price
- Key insights such as recent change and current trend direction

# Tech Stack

- Python
- Streamlit
- yfinance
- Pandas
- Matplotlib
- Scikit-learn
